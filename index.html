Bienvenido al Proyecto de Análisis de Eventos Culturales

Este proyecto tiene como objetivo estudiar y analizar los datos de asistencia a eventos organizados por la Fundación Juan March durante la temporada cultural 2023/2024 en Madrid. A través de una metodología rigurosa basada en la minería de datos y en técnicas estadísticas, se han extraído patrones de asistencia, se han limpiado y depurado los datos, y se han construido modelos predictivos para anticipar tendencias futuras.

Fases del Proyecto

  • Extracción de datos: conexión segura con bases de datos MongoDB y selección de datos relevantes.
  • Limpieza de datos: filtrado de eventos cancelados, depuración de entradas inconsistentes y preparación para análisis.
  • Visualización de datos: representación gráfica para identificar patrones y comportamientos clave.
  • Modelos de predicción: clustering de actos y series temporales para anticipar la evolución de la asistencia.

Resumen Técnico

Se trabajó principalmente con las bases de datos culturales (127,77 MB) y reservas (57,00 MB), extraídas de un ecosistema global que ocupa 23,82 GB de información estructurada. La colección de actos culturales contiene 289 eventos mientras que las validaciones de asistencia registran 282 entradas, combinadas exitosamente en 246 registros íntegros para el análisis.

Herramientas Utilizadas

El desarrollo completo del proyecto de análisis de datos culturales se ha apoyado en una serie de herramientas y tecnologías seleccionadas por su robustez y adecuación a las necesidades del tratamiento de datos masivos y su análisis posterior.

  • Python: lenguaje de programación principal utilizado para el tratamiento de datos y modelado.
  • Jupyter Notebook: entorno de desarrollo interactivo donde se estructuraron todas las fases del proyecto.
  • Pandas: biblioteca esencial para la manipulación, limpieza y transformación de datos.
  • Pymongo: conector para acceder y extraer información desde bases de datos MongoDB.
  • Matplotlib y Seaborn: librerías de visualización gráfica empleadas para representar los resultados de manera visual.
  • Scikit-learn: librería de machine learning utilizada para el modelado de clusters mediante k-Means.
  • Statsmodels: utilizada para la construcción de modelos ARIMA en el análisis de series temporales.
  • Facebook Prophet: modelo predictivo empleado para analizar tendencias estacionales a largo plazo.

Cada herramienta ha sido aplicada cuidadosamente en su fase correspondiente, garantizando un flujo de trabajo sólido y coherente que respeta las buenas prácticas tradicionales en proyectos de análisis de datos.
