Extracción de Datos

En esta sección se describe el proceso de extracción de datos desde la base de datos en MongoDB. Se han utilizado herramientas como pymongo para conectar con la base de datos y realizar consultas específicas que permiten obtener la información relevante para el análisis.

Paso 1: Configuración del entorno

Para trabajar con MongoDB en Python es necesario instalar y configurar la biblioteca pymongo. 

!pip install pymongo
import pymongo
print(pymongo.__version__)

Paso 2: Conexión a la base de datos

Se utiliza la URI del servidor y las credenciales de acceso para conectarse a la base de datos de MongoDB. 

username = "pedro"
password = "********"
mongo_uri = f"mongodb://{username}:{password}@srvmongo.fmarch.es:27017"
myclient = pymongo.MongoClient(mongo_uri)

Paso 3: Selección de las bases de datos

Se seleccionan las bases de datos sobre las que se va a trabajar:

mydb1 = myclient["culturales"]
mydb2 = myclient["reservas"]

Paso 4: Selección de las colecciones

Cada base de datos contiene una o varias colecciones. Aquí se seleccionan las colecciones específicas: 

mycol1 = mydb1["actos"]
mycol2 = mydb2["resumenv2"]

De esta forma, ya se tiene acceso a los datos sobre los que se desea trabajar.
