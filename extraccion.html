Extracción de Datos

En esta sección se describe el proceso de extracción de datos desde la base de datos en MongoDB. Se han utilizado herramientas como pymongo para conectar con la base de datos y realizar consultas específicas que permiten obtener la información relevante para el análisis.

Paso 1: Configuración del entorno

Para poder acceder a los datos almacenados en MongoDB desde Python, es necesario preparar el entorno de trabajo. Esto implica la instalación de la biblioteca pymongo, que permite establecer conexiones con bases de datos MongoDB, realizar consultas y gestionar la información desde scripts de Python.

Paso 2: Conexión a la base de datos

La conexión con la base de datos se realiza utilizando una URI que incluye las credenciales de acceso (usuario y contraseña), el host del servidor y el puerto correspondiente. Esta configuración permite acceder de forma segura a las colecciones de datos almacenadas.

Paso 3: Selección de las bases de datos

Una vez establecida la conexión, se seleccionaron las bases de datos relevantes para el análisis. En este caso, se trabajó con dos: una que almacena los actos culturales y otra que contiene información sobre las reservas y validaciones de entradas.

Paso 4: Selección de las colecciones

Dentro de cada base de datos existen colecciones específicas. Se seleccionaron las colecciones actos y resumenv2, que contienen respectivamente información sobre los eventos realizados y sobre las entradas validadas para dichos eventos.
